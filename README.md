## 🛠 How to Use

1. **Clone the Repository**

   ```bash
   git clone https://github.com/Vishnu-Varthan1/dsa-.git
   ```

2. **Navigate into the project**

   ```bash
   cd dsa-
   ```

3. **Run any problem file** (example for C++)

   ```bash
   g++ 792-binary-search.cpp -o binary_search
   ./binary_search
   ```
